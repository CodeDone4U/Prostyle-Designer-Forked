== Installation ==

1. Upload `prostyle-designer` plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Install Woocommerce for full functionality of this plugin.
4. Permalinks must be set to postname.
